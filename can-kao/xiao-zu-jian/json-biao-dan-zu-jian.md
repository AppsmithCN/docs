# JSON表单组件

JSON 表单组件支持通过JSON 字段快速生成表单，你可以将此表单绑定到查询或 API 以即时更新数据。

| 自动生成  | 启用后，当源数据中的字段类型更改时，表单布局将自动更新。 |
